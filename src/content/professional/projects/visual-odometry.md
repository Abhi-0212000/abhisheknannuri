Programmed a feature extraction and trajectory reconstruction pipeline for autonomous vehicle localization using monocular/stereo sequential image streams.

- **[1]** Programmed a feature extraction pipeline using SIFT descriptors and FLANN-based matching, reducing outliers by 85% in sequential image processing.
- **[2]** Implemented visual odometry using Essential Matrix Decomposition, achieving 92% accuracy in camera motion estimation.
- **[3]** Reconstructed vehicle trajectory by computing incremental camera pose transformations.
