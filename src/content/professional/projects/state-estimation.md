Formulated an Extended Kalman Filter (EKF) sensor fusion pipeline integrating multi-modal sensor inputs for vehicle state estimation and tracking.

- **[1]** Formulated Extended Kalman Filter (EKF) sensor fusion pipeline integrating pre-recorded and processed IMU, GNSS, and LIDAR data for vehicle state estimation and localization.
- **[2]** Developed robust measurement update functions and error state modeling to process multi-sensor data with covariance tracking.
- **[3]** Built 3D visualization tools using Matplotlib for trajectory analysis and error validation across position and orientation states.
