Engineered a production-grade Global Path Planner in C++ using Lanelet2 and GPS-based navigation featuring fault-tolerant operation and seamless ROS2 integration.

- **[1]** Architected comprehensive modular ROS2-based system architecture with robust exception handling and logging, significantly streamlining development workflow and system maintenance.
- **[2]** Implemented Local Trajectory Planner using Quintic Bézier splines achieving C2/G2 path continuity, reducing trajectory deviations by 40%.
- **[3]** Integrated state-of-the-art velocity profiling system using Dynamic Window Approach (DWA), enhanced to satisfy robot's kinematic and dynamic constraints.
- **[4]** Crafted a custom RPROP optimization framework with gradient-based mapping, reducing trajectory traversal time by 25% compared to human-defined centerlines.
